# Backend for Portfolio Webpage

### `GET` - get all cards
- `/api/projectcards`

### `GET` - get card by id
- `/api/projectcards/id`

### `POST` - add a card
- `/api/projectcards`

### `DELETE` - delete by id
- `/api/projectcards/{enter id}`

### `UPDATE` - update by id
- `/api/projectcards/{enter id}`
